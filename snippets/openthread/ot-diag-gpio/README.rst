.. _snippet_ot_diag_gpio:

OpenThread snippet for diagnostic GPIO commands (ot-diag-gpio)
==============================================================

.. contents::
   :local:
   :depth: 2

Overview
********

This snippet enables the :kconfig:option:`CONFIG_GPIO_GET_DIRECTION` option to support querying the GPIO direction.
This snippet is used in the OpenThread samples to configure the DK buttons and LEDs for diagnostic GPIO commands.
